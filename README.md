# Aircraft Engine Monitoring System

## Description
Projet IoT de surveillance des moteurs d’avion utilisant ESP32, MQTT et Node-RED.

## Technologies
- ESP32
- MQTT
- Node-RED
- DHT22
- Dashboard Node-RED

## Fonctionnalités
- Surveillance température EGT
- Pression d’huile
- Vibrations moteur
- Fuel Flow
- Dashboard temps réel
